YourStats is a Ruby on Rails application that allows users to upload individual responses to a website
and view those responses displayed in text and graphical formats. 
This project shows what can be accomplished by combining many different web based development tools,
including Ruby on Rails, HighCharts, the Bootstrap framework, and a PostgreSQL database. 
It was developed using Ubuntu  and RubyMine, 
utilizing git version control hosted by GitHub and built and hosted using Heroku.
